## args_config需要配置的路径:

all_pdfpath:所有的pdf路径

pdfpath：你想处理的pdf路径

fileSavePath：pdf切下来的图片、处理过的json文件、layout的结果存放的位置

picTableSavePath：用户需要的图片和表格存放的位置

all_filePath：所有的处理过的pdf的结果存放的位置

Result_json：最终大json存放的位置

paddle_model_path:paddle模型的位置
